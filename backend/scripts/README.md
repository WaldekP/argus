# Skrypty backendu

Tu trafią skrypty seedujące bazę mediów i dziennikarzy:

- scraping publicznych bio i materiałów dziennikarzy/redakcji,
- analiza LLM (profile stylu, playbooki rozmowy),
- wynik zapisywany jako pliki seed (SQL/JSON) używane przy migracjach.

Skrypty uruchamiamy ręcznie/z CI, nigdy z klienta aplikacji.
